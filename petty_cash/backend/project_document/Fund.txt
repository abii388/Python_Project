Fund Module Documentation
Overview

A fund contains:
================
fund name
description
total budget
assigned managers (cashiers)
timestamps

This module supports role-based access control.

Roles and Permissions
=======================
1. Admin
Full system access.

Can:
Create funds
View all funds
Update all funds
Delete all funds
Assign managers

2. Cashier
Fund manager.

Can:
View assigned funds only
Create funds
Update assigned funds only
Delete assigned funds only

When a cashier creates a fund:
the cashier is automatically added as manager

3. Auditor
Read-only access.

Can:
View all funds

Cannot:
Create
Update
Delete

4. Approver
Currently no fund permissions.

This role will be used later for:
transaction approvals
reimbursement approvals
expense approvals
